# Flash
Flash is a web based code execution platform.

## Installation

Get packages

```bash
npm install
```

## Usage

```bash
cd /client
npm run dev
 -- Open the specified url in browser which will most likely be "localhost:3000"

cd /server
npm run dev
```
### Built With

* Frontend: Next.js
* Backend: Express, Node.js
* Database: Mongodb
